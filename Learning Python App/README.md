# Python Party — a fun, funny terminal tutor

This is a minimal rich-text terminal app that teaches Python with short lessons, runnable snippets, and quizzes.

Quickstart

```bash
python3 -m pip install -r requirements.txt
python3 teach.py        # interactive mode
python3 teach.py --demo # autoplay first lesson
```

Files

- `teach.py` — main application (interactive lessons, runner, quizzes)
- `requirements.txt` — runtime dependency: `rich`
